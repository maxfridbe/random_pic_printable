# Random Pic Printable

A simple web application that generates a 4x5 grid of random images (0.7" x 1.0" each) with 1" spacing, optimized for printing on letter-size paper.

## Live Demo
[https://maxfridbe.github.io/random_pic_printable/](https://maxfridbe.github.io/random_pic_printable/)

## Features
- **Randomized Images:** Uses Picsum Photos to fetch random images.
- **Print Optimized:** Automatically hides UI elements when printing.
- **Grid Layout:** Precise 4x5 grid layout with 1-inch spacing.
- **Dimensions:** Specifically designed for 0.7" x 1.0" thumbnail prints.

## How to Use
1. Open the [Live Demo](https://maxfridbe.github.io/random_pic_printable/).
2. Click **Randomize** to get a new set of images.
3. Click **Print Sheet**.
4. In the print dialog, set margins to **"None"**.
